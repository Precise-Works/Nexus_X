# Nexus_X
A polished, responsive static landing page showcasing event details, speakers, partnerships, and a countdown timer. Built with semantic HTML, modern CSS, and vanilla JavaScript for lightweight performance.

Project

Name: NEXUS X — Event landing page for AI & Blockchain summit.
Overview

Description: A polished, responsive static landing page showcasing event details, speakers, partnerships, and a countdown timer. Built with semantic HTML, modern CSS, and vanilla JavaScript for lightweight performance.
Features

Hero: Video background with clear call-to-action buttons.
Countdown: Live event countdown timer.
Speakers: Responsive speaker cards and bios.
Partnerships: Partnership/sponsorship section with responsive layout.
Progressive UX: Mobile-first responsiveness and smooth in-page scrolling.
Getting Started

Run locally: Open index.html in your browser — no build step required.
Edit content: Update HTML text and replace assets in the vid and image files.
Files

HTML: index.html — main markup.
Styles: index.css — site styles and responsive rules.
Scripts: script.js — interactions (countdown, nav, smooth scroll).
Assets: vid — video and media assets.
